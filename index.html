VIONIX — Performance & Privacy

A lightweight Android utility that gives you secure DNS routing (local VPN), storage insights (apps, images, video, audio), and on-device performance tools. Minimal, private, and built for power users.

Download (Releases) Contact

DNS Guard (VPN)

Local DNS routing via Android VPN (DNS-only). Persistent notification while active.

Storage Analyzer

Scan & preview apps, photos, videos and audio. Media access is runtime-only and local-only.

Performance Tools

Lightweight optimizations, weekly summaries, and app suggestions — processed on-device.

Privacy-first

No analytics. No data upload by default. Clear permissions and user control.
